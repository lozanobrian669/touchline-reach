# Touchline Reach

Touchline Reach is a visual tool for customizing and testing the ball interaction range within Touchline. It includes a simple interface, saveable settings, and options designed for comfortable use on PC or mobile.

## Features

### Home

- Displays general information about the script.
- Includes a changelog with the latest updates.
- Adds project and community credits.

### Reach

- Lets you adjust the ball interaction range.
- Automatically calculates the hitbox height based on the chosen reach.
- Shows a translucent debug box to visualize the range in real time.
- Saves the configured value for reuse in future sessions.

### Speed

- Lets you adjust movement speed.
- Includes support for actions like Tackle and Dive.
- Helps test mobility behavior during fast plays.

### Configuration

- Auto-save configuration.
- Accent color picker.
- Available visual themes: Nebula, Midnight, Carbon, and Abyss.
- Language selector: English / Español.
- Option to pin the floating Nebula button.
- Button to manually save the configuration.

### Community

- Includes access to the official Discord.
- Copies the invite link to the clipboard when the executor allows it.

## Usage

Run the main script from the repository:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/lozanobrian669/touchline-reach/refs/heads/main/touchline-v1.lua"))()
```
