# Touchline Reach

Touchline Reach es una herramienta visual para personalizar y probar el rango de interacción con el balón dentro de Touchline. Incluye una interfaz simple, ajustes guardables y opciones pensadas para usar el panel de forma cómoda en PC o celular.

## Funciones

### Inicio

- Muestra información general del script.
- Incluye changelog con las últimas novedades.
- Agrega créditos del proyecto y la comunidad.

### Reach / Rango

- Permite ajustar el alcance de interacción con el balón.
- Calcula automáticamente la altura de la hitbox según el alcance elegido.
- Muestra una caja de depuración translúcida para visualizar el rango en tiempo real.
- Guarda el valor configurado para reutilizarlo en futuras sesiones.

### Speed / Velocidad

- Permite ajustar la velocidad de movimiento.
- Incluye soporte para acciones como Tackle y Dive.
- Ayuda a probar el comportamiento de movilidad durante jugadas rápidas.

### Configuration / Configuración

- Autoguardado de configuración.
- Selector de color de acento.
- Temas visuales disponibles: Nebula, Midnight, Carbon y Abyss.
- Selector de idioma: English / Español.
- Opción para fijar el botón flotante de Nebula.
- Botón para guardar la configuración manualmente.

### Community / Comunidad

- Incluye acceso al Discord oficial.
- Copia el enlace de invitación al portapapeles cuando el ejecutor lo permite.
